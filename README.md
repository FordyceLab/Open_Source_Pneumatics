# Open_Source_Pneumatics
An Open-Source, Fully Programmable Pneumatic Setup for Universal Microfluidic Operation and Automated Control

This repository includes Build Guides (All Modules), Build Description and Overview (Manuscript), Customization Options (see Figures), all Design Files (Design File Folder) and a Bill of Materials for 3 build options of an open source microfluidic pneumatic controller. 

We have described this controller in our BioArXiv release which can be found here: 

http://www.biorxiv.org/content/early/2017/08/13/173468

DOI: https://doi.org/10.1101/173468
